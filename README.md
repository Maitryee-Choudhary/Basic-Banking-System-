# Basic-Banking-System-

## Introduction
>  
> A web application used to transfer money between two users.

## Technologies Used:
>  ### Front-End:
- HTML
- CSS
- Bootstrap
> ### Back-End:
-  PHP 
-  MYSQL

### Flow of the website
Home > View Customer > Select One Customer > Transfer Money > Select Receiver > View Customer
